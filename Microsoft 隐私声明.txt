Windows
Windows 是个性化的计算环境，可使你跨计算设备无缝漫游和访问各种服务、首选项和内容（从手机到平板电脑再到 Surface Hub）。不同于驻留在设备上的静态软件程序，Windows 的关键组件以云为基础，并且 Windows 的云和本地要素定期更新，从而为你提供最新的改进和功能。为了提供这类计算体验，我们收集与你、你的设备以及你如何使用 Windows 有关的数据。而由于 Windows 是专属于你的，我们也向你提供我们收集的你的个人数据以及我们使用你个人数据的方式，供你选择。请注意，如果你的 Windows 设备由组织机构管理（例如你的单位或学校），你的组织机构可能会使用 Microsoft 或其他公司提供的集中管理工具访问和处理你的数据，以便控制设备设置（包括隐私设置）、设备策略、软件更新、我们或组织机构进行的数据收集或你设备的其他方面。此外，你的组织可能会使用 Microsoft 或其他公司提供的管理工具来访问和处理来自该设备的数据，包括你的交互数据、诊断数据以及通信和文件内容。

Windows 设置（以前称为“电脑设置”）是 Microsoft Windows 的重要组件。它为调整用户首选项、配置操作系统和管理已连接设备提供了一个方便的界面，让你可以管理用户账户、调整网络设置和个性化 Windows 的各个方面。Windows 为应用访问各种设备功能(如设备的摄像头、麦克风、位置、日历、联系人、历史记录、信息等)提供了一种机制，同时控制对个人数据的访问权限。在 Windows 设置中，每种功能都有自己的隐私设置页面，因此你可以控制哪些应用可以使用每种功能。以下是“设置”的一些主要功能:

自定义：你可以个性化设置 Windows 的各个方面，包括外观、语言设置和隐私选项。Windows 设置会在控制音量时使用麦克风，在使用集成摄像头时使用摄像头，在夜间更改亮度时使用位置，以帮助你自定义 Windows。
外围设备管理：安装和管理外围设备，如打印机、显示器和外部驱动器。
网络配置：调整网络设置，包括 Wi-Fi、以太网、移动电话网络和 VPN 连接，如果设备支持移动电话网络，将使用物理 MAC 地址、IMEI 和移动电话号码。
帐户管理：添加或移除用户帐户、更改帐户设置以及管理登录选项。
系统级选项：配置显示设置、通知、电源选项、管理已安装应用列表等。
隐私&安全管理：配置隐私偏好设置，例如位置、诊断数据收集等。通过开启或关闭单个应用和服务，对其访问设备功能的权限进行微调。
有关 Windows 中的数据收集的详细信息，请参阅  Windows 的数据收集摘要。本声明讨论了 Windows 10 和 Windows 11，并且此部分中对 Windows 的引用与这些产品版本相关。Windows 的早期版本（包括 Windows Vista、Windows 7、Windows 8 和 Windows 8.1）遵循其自己的隐私声明。
